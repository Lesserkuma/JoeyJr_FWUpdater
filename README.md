# Firmware Updater for Joey Jr for macOS

To install the Joey Jr firmware for [FlashGBX](https://github.com/lesserkuma/FlashGBX) on macOS, enter this command in a Terminal window:

`curl -sSL https://raw.githubusercontent.com/lesserkuma/JoeyJr_FWUpdater/main/dl.sh | bash`

Your “BENNVENN” drive will disappear after the upgrade, this is normal. The FlashGBX software can be used to restore the Drag'n'Drop firmware later.

This is only for macOS. Users of other operating systems can use FlashGBX directly: Options → Tools → Firmware Updater → Joey Jr.
